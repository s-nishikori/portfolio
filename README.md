## Getting Started

1. 実行環境の構築: プロジェクトのルートディレクトリで、以下のコマンドを実行してDockerコンテナを起動する

```
docker compose up -d
```

2. 開発サーバーの起動

```bash
npm run dev
```

3. [http://localhost:3000](http://localhost:3000) にアクセス
